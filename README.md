# Sneaker-Loop

Sneaker-Loop is a website for sneakerheads. Right now, it focuses on keeping you up-to-date with the latest sneaker releases.

## Features

- **Releases Page**: Browse upcoming and past sneaker drops with all the details..
- **Feedback Page**: Share your thoughts and read feedback from other sneaker enthusiasts.

## Directory Structure

- `html/`: Contains the HTML files for the website's pages.
  - `home.html`: The main landing page.
  - `releases.html`: Page for sneaker release information.
  - `feedback.html`: Page for user feedback.
- `css/`: Contains the CSS files for styling the website.
  - `home.css`: Styles for the home page.
  - `releases.css`: Styles for the releases page.
  - `feedback.css`: Styles for the feedback page.
  - `layout.css`: General layout styles shared across pages.
- `script/`: Contains JavaScript files for interactive functionality.
  - `sl_formsubmit.js`: Handles form submissions on the feedback page.
- `images/`: Contains images used on the website.
  - `shoe release pictures/`: Images related to sneaker releases.

## How to Use

1. Open the `home.html` file in your browser to start exploring the website.
2. Navigate to the `Releases` page to check out the latest sneaker drops.
3. Visit the `Feedback` page to share your opinions and connect with the community.
